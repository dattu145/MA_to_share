1). Use different selection and transform tools to modify or improve an image using image editing tools like GIMP/ InkScape / Krita

Aim :

To learn and apply different selection and transform tools in an image editing software (GIMP / Inkscape / Krita) to enhance or modify an image creatively and effectively.

Principle :

Selection tools allow specific parts of an image to be isolated for editing, while transform tools enable resizing, rotating, flipping, or skewing selected elements to achieve the desired visual effect.

Apparatus Required :

  • Computer with image editing software installed (GIMP / Inkscape / Krita)
  • Sample image file
  • Mouse or graphic tablet (optional)

2). Create logos and banners for home pages of websites using image editing tools like GIMP /InkScape / Krita

Aim :

To design visually appealing logos and banners for website homepages using image editing tools such as GIMP, Inkscape, or Krita.

Principle :

Logos and banners form the first visual impression of a website. Using image editing tools, one can create professional and responsive graphics by applying vector and raster-based editing, typography, gradients, layer styling, and export settings suitable for web formats.

Apparatus Required :

  • Desktop/Laptop with GIMP / Inkscape / Krita installed
  • Input devices: Mouse or Graphics Tablet (optional)
  • Reference images or sketches
  • Fonts, icons, and color palette

Procedure :

  1. Open image editing tool (e.g., GIMP).
  2. Set canvas size suitable for logo (e.g., 800x800 px) and banner (e.g., 1280x400 px).

4). Create an audio file and generate an audio compression codec like MP3.

Aim :

To create an audio file and compress it using an audio codec like MP3 to reduce file size while maintaining acceptable audio quality.

Principle :

Audio compression reduces the size of digital audio files using algorithms called codecs (COder-DECoder). MP3 (MPEG-1 Audio Layer III) is a popular lossy compression format that removes inaudible sound frequencies to significantly reduce file size with minimal loss in quality.

Apparatus Required :

  • Desktop/Laptop with Audacity, FFmpeg, or any audio converter tool installed
  • Microphone (for recording) or pre-recorded audio clips
  • MP3 encoder (LAME for Audacity)
  • Headphones for monitoring audio quality

Procedure :

  1. Open Audacity and import or record an audio file.
  2. Perform basic editing: noise reduction, normalization, trimming silence.
  3. Go to File → Export → Export as MP3.
  4. Choose bitrate (128 kbps is standard), sample rate (44.1 kHz).
  5. Save the file with .mp3 extension using the LAME MP3 encoder.
  6. Optionally, use FFmpeg:
    7. ffmpeg -i input.wav -b:a 128k output.mp3
  7. Compare file size before and after compression.

Result :

Successfully created and compressed an audio file into MP3 format using an audio codec, achieving over 80% reduction in file size with good quality retention.
