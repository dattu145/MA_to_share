1). Use different selection and transform tools to modify or improve an image using image editing tools like GIMP/ InkScape / Krita

Aim :

To learn and apply different selection and transform tools in an image editing software (GIMP / Inkscape / Krita) to enhance or modify an image creatively and effectively.

Principle :

Selection tools allow specific parts of an image to be isolated for editing, while transform tools enable resizing, rotating, flipping, or skewing selected elements to achieve the desired visual effect.

Apparatus Required :

  • Computer with image editing software installed (GIMP / Inkscape / Krita)
  • Sample image file
  • Mouse or graphic tablet (optional)

Table :

Tool Name Category Function
Rectangle Select Selection Selects rectangular regions
Free Select Selection Selects irregular or custom-shaped regions
Fuzzy Select Selection Selects contiguous regions based on color
Move Tool Transform Moves a selected area or layer
Scale Tool Transform Resizes the selected area
Rotate Tool Transform Rotates the selection or layer
Flip Tool Transform Flips the image horizontally or vertically

  1. Open the image using the software.
  2. Use the Rectangle Select Tool to select a portion of the image (e.g., a personâ€™s face).
  3. Apply Scale Tool to resize the selected portion.
  4. Use Rotate Tool to rotate an object (e.g., an object tilted 45 degrees).
  5. Use Flip Tool to mirror a selected region.
  6. Combine Free Select Tool and Move Tool to isolate and relocate objects.
  7. Export the modified image in preferred format (.jpg, .png, etc.)

Result :

Successfully used multiple selection and transformation tools to modify the image by resizing, rotating, flipping, and repositioning different elements.
